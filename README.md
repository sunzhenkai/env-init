# Descript
Tools for initialize your favorite environment.

# Install
```shell
git clone git@github.com:sunzhenkai/env-init.git
cd env-init
./activate
```
# Usage
```shell
ii cmake       # install cmake
ii python3 -b  # install python3 form source code
```
